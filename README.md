Theme for oh-my-zsh
===================

This is a theme made to be used with vi-mode plugin. The default vi mode is NORMAL.

Install
-------

- Drop the `syl20bnr.zsh-theme` in your `~/.oh-my-zsh/themes` directory.
- In your `~/.zshrc` add the following line.

        ZSH_THEME="syl20bnr"
